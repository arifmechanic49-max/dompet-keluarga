⚙️ Hubungkan ke Google Sheets

Masukkan URL Google Apps Script Anda. Setup ini hanya dilakukan sekali per perangkat.

1. Buka script.google.com → New Project → paste kode Apps Script
2. Isi SHEET_ID dengan ID Spreadsheet Anda
3. Deploy → New Deployment → Web App
   Execute as: Me | Who has access: Anyone
4. Copy URL → paste di bawah ini

URL dan nama hanya tersimpan di browser ini (localStorage). Data tersimpan aman di Google Sheets Anda.

Ganti Nama Pengguna

Nama ini akan muncul di kolom "Dicatat oleh" pada setiap transaksi.
